# Data Safety Declaration

This document summarizes the data collection and security practices for the Google Play Data Safety section.

## Data Collection
- **Location**: Approximate and Precise location (for maps and destination guidance).
- **Personal Info**: Name, Email address, User IDs (for account creation).
- **Photos/Videos**: Optional (if user uploads profile pictures or reviews).
- **App Activity**: App interactions and crash logs.

## Security Practices
- **Data Encrypted in Transit**: All data is transferred over secure HTTPS connections.
- **Data Deletion**: Users can request that their account and associated data be deleted.
